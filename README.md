# CongoPlace

CongoPlace est une maquette de site de petites annonces pour la Republique du Congo.

## Fonctionnalites

- Recherche d'annonces par mot-cle, ville, categorie et prix
- Villes disponibles: Pointe-Noire, Brazzaville, Dolisie, Nkayi
- Prix affiches en FCFA
- Creation de compte en local
- Publication d'annonce en local
- Favoris
- Carrousel d'accueil
- Cartes d'annonces avec images

## Lancer le projet

Ouvrir simplement `index.html` dans un navigateur.

Cette version est une maquette statique en HTML, CSS et JavaScript. Les donnees creees par l'utilisateur sont stockees dans le navigateur avec `localStorage`.
